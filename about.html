About Kibbo

Consumer-protection software, built in the open by one independent developer.

Kibbo is a small studio of consumer-protection tools — Chrome extensions and web apps — built by Carlos Lopez, an independent developer. Every product targets one specific way the modern web tries to trick you: fake discounts, dark patterns, phishing messages, abusive contract terms, and manipulative cookie banners.

The idea started with a simple frustration. Online stores and services have quietly turned everyday browsing into a minefield of small manipulations — countdown timers that secretly reset, "original" prices that were never real, reject buttons hidden three clicks deep. Regulators write rules about these tactics, but they can't inspect millions of product pages in real time. Software running in your own browser can.

What Kibbo builds

Each tool does one job well and runs locally on your machine wherever possible, so your browsing data stays with you. The flagship, Web Guardian, bundles eight protection modules into a single extension. The others — a phishing detector powered by Claude AI, a price-history tracker, a dark-patterns detector, a contract analyzer, a checkout shield, and a cookie-consent equalizer — can each be used on their own.

Source code included, no subscriptions

Kibbo sells software the old-fashioned way: you pay once and you own it. Every purchase includes the full commercial source code, so you can audit exactly what's running in your browser, modify it, or build your own tools on top of it. There are no subscriptions and no recurring fees.

Built in the open

The Kibbo blog documents how these tools actually work — the real code, the architecture decisions, the bugs, and the test results — in plain technical English. If you want to see what the developer behind Kibbo is shipping, the code lives on GitHub.
